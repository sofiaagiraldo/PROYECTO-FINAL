ARCHIVO: RutaOptima.pbix
========================

Este informe se genera AUTOMÁTICAMENTE si instalaste las dependencias del proyecto.

PASOS (primera vez en tu PC)
----------------------------

1. Abrir terminal en la carpeta del proyecto (donde está main.py).

2. Instalar:
     python3 -m venv .venv
     source .venv/bin/activate
     pip install -r requirements.txt

3. Generar datos + .pbix + ventana Tkinter:
     chmod +x run_gui.sh
     ./run_gui.sh

4. Comprobar que existe:
     powerbi/RutaOptima.pbix

5. Ver dashboard Power BI:
   - Instalar Power BI Desktop: https://powerbi.microsoft.com/desktop/
   - Abrir RutaOptima.pbix (doble clic o botón 5 en Tkinter)
   - Seguir la sección 4 de INSTRUCCIONES_LOCAL.md para las 4 gráficas

Si el .pbix NO aparece
----------------------
  source .venv/bin/activate
  pip install pbix-mcp
  python -c "from Backend.generar_pbix import asegurar_archivo_pbix; from Backend.config import ruta_base_datos; print(asegurar_archivo_pbix(ruta_base_datos()))"
